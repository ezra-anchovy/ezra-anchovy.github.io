Confirmation step by design

QRion is built around inspection before action — including an explicit confirm-open moment.

Important: QRion is not a guarantee that every destination is safe. It provides on-device signals (blocklist checks, URL shortener detection, domain/path heuristics) so you can make a better decision.

Why use this instead of the iPhone camera?

Because the camera helps you open a QR code. QRion helps you inspect it first.
