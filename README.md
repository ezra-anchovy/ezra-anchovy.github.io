# Anchovy Labs Website

Static company portfolio site for Anchovy Labs.

## Pages
- `index.html` - landing page with featured apps
- `apps.html` - app portfolio cards
- `about.html` - team story and values
- `privacy.html` - privacy policy and regulations summary
- `support.html` - FAQ and support contact

## Features
- Shared stylesheet (`styles.css`)
- Dark mode by default with light-mode toggle
- Mobile-first responsive layout
- Scroll reveal animations with IntersectionObserver
- Accessible semantic HTML5

## Run Tests
```bash
python3 -m pytest test_website.py -v
```
